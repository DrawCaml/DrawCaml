type dlayout =
| FloatLayout
| GridLayout
| Other

Different types of layout:

  • The float layout allows you to place elements by giving their exact position in the layout
  • The grid layout allows you to place elements in a grid by giving their coordinates in the grid

Note that the use of Other is deprecated.
